# 🚀 CafeManager — Система автоматизации кафе
📖 Описание проекта
CafeManager — это интеллектуальная информационная система, разработанная для полной автоматизации бизнес-процессов заведения общественного питания (ООО «Кофе-Хауз»). Проект помогает объединить управление заказами, меню, складским учетом и персоналом в единый цифровой контур, сокращая время обслуживания гостей и сводя к минимуму риск ошибок. ☕️

✨ Основные возможности
📱 Управление заказами: Прием, маршрутизация и отслеживание статусов блюд (KDS) в режиме реального времени.

🪑 Схема зала: Интерактивное управление рассадкой гостей.

🍲 Управление меню и складом: Гибкое редактирование цен, состава блюд и автоматический контроль остатков.

👤 Ролевая модель: Надежное разграничение прав доступа (Администратор, Официант, Повар, Бариста).

📊 Аналитика: Модуль отчетов по выручке, популярности позиций и эффективности работы персонала.

🛠 Стек технологий
Backend: .NET 8 (Web API) 🖥️

Frontend (Клиентская часть): C# WPF (MVVM) 🖥️

База данных: MS SQL Server 2022 🗄️

Проектирование: UML, ER-моделирование 📐

🏗 Архитектура базы данных
Система использует мощную реляционную БД с поддержкой:

Ссылочной целостности: FOREIGN KEY с правилами каскадного удаления.

Оптимизации: Составные индексы, хранимые процедуры и Views.

Конкурентности: Уровень изоляции READ COMMITTED SNAPSHOT для быстрой работы без блокировок.

🚀 Установка и запуск
Подготовка БД: Разверните SQL-скрипт из папки /database в вашем экземпляре MS SQL Server. ⚙️

Запуск API: Соберите проект серверной части в Visual Studio 2022.

Запуск клиента: Откройте WPF-приложение и используйте учетные данные администратора для входа. 🔑

🧪 Тестирование
Unit-тестирование: Проведено с использованием xUnit (покрытие критических модулей — 78%). ✅

Нагрузочное тестирование: Эмуляция 100+ одновременных сессий показала стабильную работу API с откликом 18–45 мс. ⚡️
